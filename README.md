# Student Management System

## Description

Student Management System is a simple Python-based mini project used to manage student records. It allows users to add, view, search, and delete student details. This project is useful for beginners to understand basic Python concepts like lists, functions, loops, and conditions.

## Algorithm

1. Start the program.
2. Create an empty list to store student records.
3. Display the menu options:

   * Add Student
   * View Students
   * Search Student
   * Delete Student
   * Exit
4. Take user choice as input.
5. If user selects **Add Student**:

   * Enter name, roll number, and marks.
   * Store details in the list.
6. If user selects **View Students**:

   * Display all student records.
7. If user selects **Search Student**:

   * Enter roll number.
   * Check if the student exists.
   * Display details if found.
8. If user selects **Delete Student**:

   * Enter roll number.
   * Remove the student record if found.
9. Repeat until the user chooses Exit.
10. Stop the program.

## Features

* Add new student records
* View all student details
* Search student by roll number
* Delete student records
* Simple menu-driven program
* Easy to understand and use
